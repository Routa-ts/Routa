# create-routa-ts

Project scaffolder for Routa, a schema-first, OpenAPI-aware REST framework for new TypeScript APIs.

Use this package through your package manager's create command to generate a Routa starter project with TypeScript, Biome, route contracts, and optional OpenAPI scaffolding.

## Usage

```sh
pnpm create routa-ts@latest
```

You can also run the package directly:

```sh
npm create routa-ts@latest
```

Generated projects require Node.js 24+ and work with npm, pnpm, Yarn, Bun, or another
compatible package manager. The scaffold does not lock the project to one package manager.

## What It Creates

- TypeScript Routa app structure
- Route contract examples
- Biome configuration
- Scripts for checking, building, and running the API
