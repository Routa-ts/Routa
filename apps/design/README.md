# Routa Design Lab

Parallel Astro + Starlight app for reviewing Routa visual identity.

This is **not** the public docs site (`apps/docs`). Docs ship **Lattice** at
`https://routa-ts.dev` with a temporary stub at `/` and documentation under `/docs/`.

## Commands

| Command | Action |
| --- | --- |
| `pnpm --filter @routa-ts/design dev` | Dev server on port **4322** |
| `pnpm --filter @routa-ts/design build` | Build to `apps/design/dist` |
| `pnpm --filter @routa-ts/design preview` | Preview the build |
