## Field notes · Asides

<Aside type="note" title="Note · Contour line">
	`src/routes/users/$userId/posts/route.ts` is the trail to `POST /users/:userId/posts`.
</Aside>

<Aside type="tip" title="Tip · Pack light">
	Keep handlers thin — business logic lives off the trail in app modules.
</Aside>

<Aside type="caution" title="Caution · Unmarked fork">
	Route groups like `(private)` change middleware, not the public path. Label them clearly.
</Aside>

<Aside type="danger" title="Danger · Off-trail edit">
	Hand-patching generated OpenAPI erases the map on the next regenerate.
</Aside>

## Expedition steps

<Steps>

1. Stake the trailhead.

	```sh
	pnpm create routa-ts@latest
	```

2. Cut a new path under `src/routes`.

	```ts
	// src/routes/users/$userId/route.ts
	export default defineRoute({
		get: createRoute({
			input: { params: UserParams },
			responses: {
				success: { status: 200, schema: UserSchema },
				notFound: { status: 404, schema: ErrorSchema },
			},
			run: async ({ input }) => users.get(input.params.userId),
		}),
	});
	```

3. Survey, then hike.

	```sh
	pnpm routa check
	pnpm routa start
	```

</Steps>

## Waypoint contract

```ts
export default defineRoute({
	get: createRoute({
		input: { params: ProjectParams },
		responses: {
			success: { status: 200, schema: ProjectSchema },
		},
		run: async ({ input, ctx }) => {
			return projects.get(input.params.projectId, ctx);
		},
	}),
});
```
