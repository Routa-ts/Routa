## Boot sequence

<Steps>

1. Scaffold.

	```sh
	pnpm create routa-ts@latest my-api
	cd my-api
	```

2. Inspect the generated entry.

	```ts
	// src/routa.ts
	import { createApp } from "@routa-ts/core";
	```

3. Run the loop.

	```sh
	pnpm routa check
	pnpm routa start
	# listening on :3000
	```

</Steps>
