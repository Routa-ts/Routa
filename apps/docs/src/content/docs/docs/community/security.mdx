title: Security
description: How to report Routa security issues.
---

Please do not report security vulnerabilities through public GitHub issues.

## Reporting

Email security reports to:

```txt
security@routa-ts.dev
```

Include enough detail to reproduce and evaluate the issue:

- affected package or workflow
- Routa version or commit
- reproduction steps
- impact
- suggested mitigation, if known

## Public Policy

Read the repository security policy for the current process:

[Routa security policy](https://github.com/Routa-ts/Routa/blob/main/SECURITY.md)
