# Routa Trademark Policy

Routa's code is available under the MIT License. The Routa name, logo, and official project identity are not licensed for confusing or misleading use.

## Allowed Use

You may use the Routa name to:

- refer to the official project
- describe compatibility with Routa
- say that your app, library, article, or package uses Routa
- link to the official Routa repository or documentation

Apps and services that use Routa as a backend framework do not need to publicly mention Routa.

## Forks and Derivatives

If you publish a fork, derivative framework, or redistributed version based on Routa, include a clear attribution such as:

```txt
Based on Routa.
```

This attribution should appear in a visible project location such as the README, documentation, package metadata, or about page.

Forks and derivatives must not imply that they are official Routa projects, endorsed by Routa maintainers, or maintained by the Routa project unless they have written permission.

## Official Use

Do not use the Routa name or logo in a way that suggests official status for:

- unrelated products
- hosted services
- distributions or forks
- commercial offerings
- package names that could be confused with official Routa packages
